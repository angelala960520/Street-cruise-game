# Street Cruise

A small browser game where you steer a car through highway traffic, collect coins, and pick up power-ups.

## Play

Open `index.html` in a browser.

## Controls

- `A`: steer left
- `D`: steer right
- `Space`: start or retry

## GitHub Pages

After pushing this repo to GitHub, enable Pages from the repository settings:

1. Go to `Settings` -> `Pages`.
2. Set the source to `Deploy from a branch`.
3. Choose the `main` branch and `/root`.
4. Save.
